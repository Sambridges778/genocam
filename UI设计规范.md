GenoCam UI 设计规范

设计理念
- 风格关键词：Soft Minimalism（柔、静、透、悬）。
- 目标体验：未来感且轻盈，通过光、层、字、色四要素建立空间与温度。

基础风格四要素
- 光（空间层次）
  - 背景使用柔和渐变与雾面：米白 → 浅紫（渐变色不超过 2 种，饱和度 < 60%）。
  - 背景模糊建议：Backdrop Blur 强度 32–48（谨慎使用超 60，易影响可读性）。
  - 适度内阴影/磨砂质感增强“呼吸感”。
- 层（体积与悬浮）
  - 卡片采用悬浮感而非实线边框分区。
  - 阴影建议：`offsetY=8vp`、`blur=20vp`、`opacity 10–16%`，随层级递增。
  - 半透明边界 + 模糊背景优先于灰色边框。
- 字（Urbanist + 极简排版）
  - 字体：优先 Urbanist。若设备不支持，使用系统无衬线作为回退。
  - 大小与层次（ArkUI 使用 `fp`）：标题 `28fp`，副标题 `18fp`，正文 `14fp`；行高正文 `20–22fp`。
  - 文本颜色：`#000000` 不同透明度，正文建议不死黑，`opacity≈80%`。
- 色（柔雾渐变 + 点缀亮色）
  - 主配色：`#B5B9C9`（浅蓝灰）、`#F6F6F6`（雾白）。
  - 点缀色：`#FAD349`（金黄），用于强调与交互反馈。
  - 渐变规范：不超过 2 种主色，避免高饱和与强对比，以维持柔和质感。

动效规范
- 节奏：慢、柔，不炫技；以 `ease-out` 为主，场景切换可用 `ease-in-out`。
- 时长建议：
  - 微交互（按钮、卡片悬浮）：160–240ms。
  - 页面转场/弹层：240–320ms。
- 叠加：透明度渐变 + 轻微位移；避免过度缩放或弹性回弹以维持沉静氛围。

布局与间距（ArkUI）
- 基础间距单位：`vp`，采用 4/8 网格系统。
  - `space-xs=4vp`、`space-sm=8vp`、`space-md=12vp`、`space-lg=16vp`、`space-xl=24vp`。
- 圆角半径：
  - 按层级递增：`radius-sm=8vp`、`radius-md=12vp`、`radius-lg=16vp`。
- 触控目标尺寸：最小 `48vp × 48vp`；相邻可点击区域保持 `8vp` 以上间距。

色彩系统与对比度
- 文本与背景对比度：正文建议满足 4.5:1，标题 3:1。
- 主题与模式：支持 `light` 与 `dark`；深色模式下保持型氛一致（降低饱和度与亮度，保留层次与透明感）。

ArkUI 资源映射（建议命名）
- 颜色（`entry/src/main/resources/base/element/color.json`）：
  - `bg_surface`（主表面）、`bg_surface_alt`（次表面）、`text_primary`、`text_secondary`、`accent_primary`。
- 字体尺寸（`float.json`，单位 `fp`）：
  - `font_size_title=28fp`、`font_size_subtitle=18fp`、`font_size_body=14fp`、`font_line_height_body=20fp`。
- 间距与圆角（可在文档中约定，ArkTS 中以常量维护）：
  - `space_*` 与 `radius_*` 如上规范。

代码使用示例（ArkTS）
- 文本：
  - `Text('正文').fontSize($r('app.float.font_size_body')).fontColor($r('app.color.text_primary')).opacity(0.8)`
- 结构卡片：
  - `Column().borderRadius(12).backgroundColor($r('app.color.bg_surface')).shadow({ radius: 20, color: '#000000', offsetY: 8, opacity: 0.12 })`

组件与状态
- 按钮：高度 `48vp`，圆角 `12vp`，主按钮使用 `accent_primary`；悬浮态提升阴影层级与轻微位移（`2–4vp`）。
- 卡片：多层次阴影与轻磨砂背景；内容区与交互区间距遵循 `space-md` 最小标准。
- 输入与开关：聚焦态使用柔和高亮边界或投影；错误态采用微红色边界与提示文案，不使用高饱和红底。

可访问性与国际化
- 文本缩放：支持系统字体缩放，不裁剪内容；标题与正文在 1.1–1.3 倍范围内仍保持版式稳定。
- 焦点提示：键盘/遥控焦点环需有清晰边界与对比度，不依赖颜色单一传达。
- 触觉反馈：关键交互（拍摄、完成、错误提示）提供轻振反馈；可在设置中开关。
- 多语言：文案长度差异下保持布局弹性；英文/中文标题均应在两行内完整展示。

图像与质感
- 颗粒与晕影：
  - 颗粒强度在内容区应适度（照片/视频预览除外），避免影响文字可读性。
  - 晕影用于大图展示或预览边界，不在信息密集区使用。
- 点光与模糊：用于强调区域或底部栏，避免覆盖主要交互控件。

规范摘要
- 保持“柔、静、透、悬”的整体调性；用光与层构筑空间，用字与色维持可读与温度。
- 使用 `vp/fp` 与 ArkUI 资源统一管理；围绕 4/8 网格和 12/16 圆角体系。
- 动效慢而有克制；组件状态清晰且符合可访问性标准。

产品界面设计（Geno 复古相机滤镜）

信息架构（IA）
- 首页（`CameraPage`）：拍照/录像预览、预设选择、附件与参数调节。
- 预设（`PresetPicker`）：胶片/数码/视频三类预设的网格选择与预览。
- 底片与重冲洗（`LabPage`）：底片库、重新冲洗（曝光/色温/颗粒等）与版本管理。
- 相册（`GalleryPage`）：已拍摄照片与视频的浏览、选择与导出/分享。
- 设置（`SettingsPage`）：网格、闪光、稳定、防抖档位、触觉反馈、存储与权限。
- 首次引导（`Onboarding`）：权限申请、手势教学、预设快速导览。

导航模式
- 主导航：底部栏（建议 4 个入口）→ `相机`、`预设`、`底片/重冲洗`、`相册`；`设置`放在页面右上角齿轮或用户头像入口。
- 二级导航：底部工具条与抽屉面板（`BottomSheet`）用于附件与参数；避免全屏遮挡预览。
- 返回与关闭：左上返回与下拉手势关闭全屏面板，维持轻盈节奏。

核心页面蓝图
- `CameraPage`
  - 顶部栏：左侧模式标签（`PHOTO/VIDEO`）、中部当前预设名、右侧设置与闪光灯切换。
  - 预览区：居中显示，支持构图网格（`grid_3x3`/`grid_phi`）与轻晕影边界，` pinch` 变焦。
  - 底部栏：左侧相册缩略图，居中快门/录制，右侧前后摄切换；下方为横向预设走马灯（可左右滑动）。
  - 侧边控制：垂直 `EV` 滑条（`-2 ~ +2`）、`ISO/快门/WB` 快捷切换按钮；长按锁定曝光。
  - 附件托盘：鱼眼/星光/色散/ND 等在底部抽屉中以卡片展示，可多选但高亮当前生效链路。
- `PresetPicker`
  - 分类页签：`胶片`、`数码`、`视频`。
  - 卡片：示例缩略图 + 名称 + 简介；点击进入详情并支持试拍预览。
  - 搜索与收藏：顶部搜索框与收藏筛选。
- `LabPage`
  - 底片库：时间线列表；进入详情后显示原片与当前版本对比（左右或上下滑动切换）。
  - 重冲洗：曝光、色温、曲线、颗粒、晕影、滤色片，支持版本快照与回退。
  - 导入：从系统相册选择图片进行处理（仅支持部分相机/预设）。
- `GalleryPage`
  - 网格布局：照片与视频缩略图，支持多选、分享、删除与导出。
  - 详情：信息面板（拍摄参数、预设链路、版本历史）与编辑入口。
- `SettingsPage`
  - 选项分组：拍摄（网格、闪光、定时）、视频（分辨率/帧率/稳定）、反馈（触觉/声音）、系统（权限/存储）。

页面布局规范（关键尺寸）
- 快门按钮：`72vp`（直径），聚焦态扩大至 `80vp` 并附轻微呼吸动效；禁用态降低不透明度至 `40%`。
- 预设走马灯项：`64vp × 64vp` 图标 + 名称，项间距 `8vp`，容器高度 `88vp`。
- 侧边滑条：宽 `32vp`，可点击区域 `48vp`；刻度间隔 `4vp`。
- 顶部栏高度：`56vp`；底部工具栏高度：`64–88vp`。
- 图标：`24vp`（常规），`20vp`（密集场景），点击目标最小 `48vp`。
- 圆角：卡片 `12vp`，抽屉 `16vp`；预览容器适配机型圆角。

组件库（建议）
- `ShutterButton`（快门）
- `ModeToggle`（照片/视频）
- `PresetCarousel`（预设横滑）
- `LensAccessoryTray`（附件抽屉）
- `EVSlider` / `ISOShutterSwitcher` / `WBSlider`
- `TimerBadge`、`FlashToggle`、`StabilizationSelector`、`GridOverlay`
- `PermissionBanner`、`ErrorToast`、`SavingIndicator`

状态与反馈
- 权限：未授权显示 `PermissionBanner`，点击触发系统权限请求；相机初始化延迟时显示加载骨架。
- 拍照：按下 → 快门动画（160–200ms）→ 保存指示器 → 成功弹出相册缩略图动画；失败显示错误 Toast。
- 录像：录制态显示时长与稳定档位，暂停/继续按钮视觉区分明显；停止后显示编码中指示。
- 双重曝光：首次拍摄后状态提示“等待第二张”，第二次后合成并给出预览与权重调节。

关键流程（简述）
- 拍照：预览 → 快门 → 相机会话捕获 → 滤镜链（LUT/曲线/颗粒等） → 媒体保存 → 底片归档。
- 录像：预览 → 录制 → 实时滤镜渲染 → 编码封装 → 媒体保存。
- 导入处理：选择媒体 → 解析 `PixelMap` → 应用预设 → 输出副本。
- 重冲洗：选择底片 → 调整参数 → 生成版本 → 保存与对比。

手势与交互
- 预设切换：左右滑动切换下一/上一预设；上滑打开预设抽屉。
- 变焦：双指 `pinch`；双击预览在预设焦距间切换（如 28/35/50）。
- 曝光：在预览区垂直拖动快速调整 EV；长按锁定曝光与对焦。
- 切摄像头：双指旋转或点击右下按钮；视频模式下显示切换动效。

适配与模式
- 明暗主题：保持色彩与层次的一致性；深色模式降低饱和度与高光强度。
- 横竖屏与比例：支持常见画幅（3:2/4:3/16:9）；控件随比例重排但避免遮挡主体。
- 性能档位：低端设备降低实时滤镜强度与预览分辨率；保持交互流畅。
